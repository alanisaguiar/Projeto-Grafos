# Projeto-Grafos
Este repositório contém o desenvolvimento de um projeto utilizando Grafos. O objetivo principal do trabalho é aplicar, de forma prática, os conceitos fundamentais da Teoria dos Grafos, por meio da implementação de estruturas, algoritmos e operações clássicas sobre grafos.

O projeto foi construído com foco em clareza conceitual, organização do código e correção algorítmica, permitindo analisar propriedades estruturais dos grafos.

## Objetivos do Trabalho
- Representar grafos de forma computacional
- Aplicar conceitos teóricos estudados em sala de aula
- Implementar algoritmos clássicos de grafos
- Analisar propriedades e comportamentos de grafos
- Desenvolver raciocínio algorítmico e lógico

## Conceitos de Grafos Abordados
O projeto trabalha, direta ou indiretamente, com os seguintes conceitos:
- Grafos direcionados e não direcionados
- Vértices e arestas
- Grau de vértices
- Caminhos e ciclos
- Componentes conexos
- Representação por matriz de adjacência e/ou lista de adjacência
- Percursos em grafos (como buscas)
- Análise estrutural do grafo

## Funcionalidades Implementadas
O sistema permite:
- Criar e manipular grafos
- Inserir vértices e arestas
- Exibir a estrutura do grafo
- Executar operações e algoritmos relacionados a grafos
- Analisar propriedades do grafo a partir da sua representação
